# GiftGuardian Home Assistant Add-on Repository

This repository contains the GiftGuardian add-on for Home Assistant.

## Add-ons provided

| Add-on | Description |
| --- | --- |
| [GiftGuardian](giftguardian/) | A personal gift tracker to manage ideas, purchases, and history. |

## Installation

1. Navigate to the **Add-ons** store in your Home Assistant.
2. Click on the **Repositories** button (three dots in the top right corner).
3. Add the URL of this repository: `https://github.com/Mikeymac75/Giftguardian`
4. Find **GiftGuardian** in the list and install it.
